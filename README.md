# GetDraft — Landing Page

Marketing landing page for **GetDraft**, the sports recruiting app where athletes get discovered.

- One self-contained `index.html` (fonts, images and scripts embedded — no build step)
- Brand: dark theme `#121212`, GetDraft green `#1FAA59`, Poppins
- 3D iPhone that rotates on scroll into the **"It's a Draft!"** match moment
- Waitlist form + book-a-call (acquisition funnel)

**Deploy:** static site — works out of the box on Vercel (no framework, no build command).
